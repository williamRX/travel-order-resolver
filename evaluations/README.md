# Evaluations Directory

Ce dossier contient les analyses détaillées des performances des modèles.

## Structure

- **`error_analysis/`** : Analyses d'erreurs
  - Faux positifs
  - Faux négatifs
  - Cas ambigus
  - Patterns d'erreurs récurrents

## Contenu

- Rapports d'évaluation détaillés
- Analyses de confusion
- Comparaisons de modèles
- Recommandations d'amélioration

## Format

- Rapports : Markdown ou HTML
- Analyses : CSV, JSON
- Visualisations : PNG, PDF
