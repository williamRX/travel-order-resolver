# Exports Directory

Ce dossier contient les modèles et artefacts prêts pour la production.

## Structure

- **`production/`** : Modèles optimisés pour production
  - Modèles quantifiés
  - Modèles optimisés
  - APIs et endpoints
  - Documentation de déploiement

## Contenu

- Modèles finaux validés
- Pipelines de preprocessing
- Scripts de déploiement
- Documentation API
- Tests de validation

## Format

- Modèles : format optimisé (ONNX, TensorFlow Lite, etc.)
- APIs : FastAPI, Flask, ou similaire
- Docker : images containerisées
