# Results Directory

Ce dossier contient tous les résultats d'évaluation et analyses.

## Structure

- **`metrics/`** : Fichiers CSV/JSON avec les métriques d'évaluation
  - Accuracy, Precision, Recall, F1-score
  - Métriques par classe
  - Historique des performances

- **`visualizations/`** : Graphiques et visualisations
  - Courbes d'apprentissage
  - Distributions
  - Comparaisons de modèles

- **`confusion_matrices/`** : Matrices de confusion sauvegardées
  - Images PNG des matrices
  - Données brutes (CSV)

## Format

- Métriques : CSV ou JSON
- Visualisations : PNG, PDF
- Matrices : PNG + CSV
